## 🎯 工作流分类

### 📦 序列号生成 (22个)
> 自动生成业务单号，支持物资和研发两大模块

| 模块 | 前缀 | 用途 |
|------|------|------|
| 物资 | JH/CG/SS | 需求计划/采购订单/入库单 |
| 物资 | CK/JH/PAY | 出库单/付款单/验收单 |
| 研发 | RD-PJ/AC/AR | 项目编号/验收编号/归档编号 |
| 研发 | RD-WK/MT/HR | 工作编号/会议编号/工时编号 |

### 📋 审批流程 (30个)
> 覆盖项目、合同、财务、安全等核心业务

| 类型 | 数量 | 流程 |
|------|------|------|
| 项目审批 | 25级 | 立项 → 执行 → 验收 |
| 合同审批 | 2级 | 起草 → 签订 |
| 财务收支 | 2级 | 申请 → 审核 |
| 安全会议 | 2级 | 提报 → 确认 |
| 物资审批 | 8类 | 计划/采购/入库/出库/验收/付款 |

### 🎯 项目生命周期 (5个)
> 完整的项目管理闭环

```
立项 → 规划 → 跟踪 → 验收 → 归档
```

### ⏰ 定时任务 (3个)
> 系统自动化运维

| 任务 | 功能 |
|------|------|
| TIMER-1 | 到期预警 (资质/合同30天内到期) |
| TIMER-3 | 安全问题汇总 (7天内安全事件) |
| TIMER-5 | 系统健康检查 (监控定时任务状态) |
